### Ответы на вопросы:

#### ❓ Какие семантические теги вы использовали?
- `<header>` - шапка сайта с навигацией
- `<nav>` - навигационное меню
- `<main>` - основное содержимое страницы
- `<section>` - секции контента
- `<article>` - статьи блога
- `<aside>` - боковая панель
- `<footer>` - подвал сайта
- `<time>` - для указания даты публикации

#### ❓ Где использовали flexbox, а где grid?
**Flexbox использован для:**
- Шапки сайта (выравнивание логотипа и навигации)
- Навигационного меню
- Боковой панели (вертикальное расположение карточек)
- Футера (распределение секций)
- Карточки "Обо мне" (центрирование контента)

**Grid использован для:**
- Основного макета страницы (2 колонки: контент + сайдбар)
- Сетки навыков в боковой панели
- Адаптивного изменения layout на мобильных

#### ❓ Какой медиазапрос добавили и что он меняет?
Добавлено 3 медиазапроса:

1. **Для планшетов (max-width: 768px):**
   - Меняет grid с 2 колонок на 1 колонку
   - Преобразует навигацию в мобильное меню
   - Изменяет layout футера

2. **Для мобильных (max-width: 480px):**
   - Уменьшает отступы
   - Изменяет сетку навыков на 1 колонку
   - Оптимизирует размеры шрифтов

3. **Для больших экранов (min-width: 1200px):**
   - Увеличивает максимальную ширину контейнера
   - Добавляет больше пространства между элементами
